# 2048 パズルゲーム

ブラウザで遊べる2048パズルゲームです。

## 遊び方

1. 矢印キーまたはWASDキーで、フィールド内のすべてのブロックを任意の方向に動かします
2. 同じ数字のブロックが衝突すると結合し、数値が2倍になります
3. 2048のブロックを作るとクリアです
4. 動かせるブロックがなくなるとゲームオーバーです

## 対応デバイス

- PC
- Mobile
